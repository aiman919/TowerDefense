\doxysubsection{Detailed Description}
Cannon tower that searches for the nearest enemy in range and fires periodically. Supports upgrades using player currency. 

Design Notes\+:
\begin{DoxyItemize}
\item Tick is disabled; uses a timer for firing logic.
\item Contains safe early exits (e.\+g., null checks for UWorld).
\item Provides public getters for Blueprint/\+UI and testing. 
\end{DoxyItemize}
