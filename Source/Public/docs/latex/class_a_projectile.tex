\doxysubsection{Detailed Description}
Simple projectile that travels in a straight line without gravity, dealing damage to enemies and the base. 

Optimizations\+:
\begin{DoxyItemize}
\item Removed empty Begin\+Play to avoid unnecessary virtual calls.
\item Safe runtime use with null checks in Init\+Projectile.
\item Inline getters available for testing and Blueprints. 
\end{DoxyItemize}
