\doxysubsection{Detailed Description}
Manages enemy wave spawning logic, including types, timing, and strength scaling. 

Features\+:
\begin{DoxyItemize}
\item Supports automatic wave launching and wave customization.
\item Dynamically chooses enemy types with probability weights.
\item Supports boss waves and strength scaling per enemy spawn. 
\end{DoxyItemize}
