Player controller for Top-\/\+Down view; allows spawning towers via mouse clicks. 

Highlights\+:
\begin{DoxyItemize}
\item No overridden Begin\+Play; clean and focused logic.
\item Public API remains clean due to encapsulated helper methods.
\item Fully supports Blueprint editing of Tower\+To\+Spawn.
