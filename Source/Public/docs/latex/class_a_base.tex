\doxysubsection{Detailed Description}
Base class with replicated health logic. 

This actor represents a destructible base with replicated health.
\begin{DoxyItemize}
\item Health and Max\+Health are initialized in the constructor.
\item Internal damage logic is separated for reusability between different damage types. 
\end{DoxyItemize}
