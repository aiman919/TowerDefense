\doxysubsection{Detailed Description}
Armored enemy type with higher HP and scaling armor. 

This enemy variant has\+:
\begin{DoxyItemize}
\item Increased base HP.
\item Armor that reduces incoming damage but guarantees at least 1 damage.
\item Armor scaling logic based on wave index. 
\end{DoxyItemize}
