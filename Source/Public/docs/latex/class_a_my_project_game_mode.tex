\doxysubsection{Detailed Description}
Custom game mode that initiates enemy waves at all spawn points when the match starts. 

Features\+:
\begin{DoxyItemize}
\item Uses static helper function to avoid hardcoded values.
\item Validates {\ttfamily Has\+Authority()} and {\ttfamily Get\+World()} to ensure logic runs only on the server.
\item Avoids code duplication in wave size logic. 
\end{DoxyItemize}
